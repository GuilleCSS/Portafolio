    Proyecto Escolar.

    Determinación de un umbral para un algorítmo de divide y venceras. Se aborda el problema de encontrar el par de puntos más cercanos mediante dos métodos: fuerza bruta y divide y vencerás. Se generan puntos aleatorios y se registran los tiempos de ejecución para ambos enfoques. Luego, se ajustan modelos matemáticos a los datos y se determina un umbral óptimo para elegir entre los métodos. Al medir el tiempo de ejecución con este umbral, se confirma que mejora significativamente la eficiencia del algoritmo.
